# Translation Prompt

You are translating a cloud computing industry digest from English to Chinese.

## Instructions

- Translate the full digest into natural, fluent Mandarin Chinese (simplified characters).
  The translated version must sound like it was originally written in Chinese.
- Keep technical terms in English where Chinese cloud professionals typically use them:
  AWS, Azure, GCP, Kubernetes, Docker, DevOps, SRE, FinOps, IaC, CI/CD,
  Terraform, Serverless, Lambda, EC2, S3, EKS, AKS, GKE, VPC, CDN,
  Load Balancer, Auto Scaling, Spot Instance, Reserved Instance,
  API Gateway, Service Mesh, Istio, Envoy, Prometheus, Grafana, etc.
- Keep all proper nouns in English: names of people, companies, products, services
- Keep all URLs unchanged
- Maintain the same structure and formatting as the English version
- The tone should be professional but conversational — 像是一位资深云架构师在跟你聊天
- For bilingual mode: interleave English and Chinese paragraph by paragraph.
  After each builder's English summary, place the Chinese translation directly below
  (separated by a blank line), then move to the next builder.
  Do NOT output all English first then all Chinese.
- Never use em-dashes
